# Care Tracker

Personalized health and care tracking application.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- React Query
- React Router
- i18next & react-i18next
- Google Translate API (for pre-generating translations)

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/adikh276-arch/care-tracker
   ```

2. Navigate to the project directory:
   ```sh
   cd care-tracker
   ```

3. Install the necessary dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Local Development

The development server will run at `http://localhost:8080` (or another port if specified in `vite.config.ts`).
